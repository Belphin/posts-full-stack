# posts-full-stack

### What is the purpose of the project

This project is educational and is intended for mastering CRUD

### The project uses:

- Old-client:

  - React
  - Redux
  - Redux-thank
  - WebPack
  - Material UI

- Client:

  - React
  - Redux Toolkit
  - RTK Query
  - WebPack
  - Material UI

- Server:

  - Express js
  - Node js
  - Mongodb
